# G-MINOR

G-MINOR is an emotionally adaptive desktop chatbot built with PyQt. It combines a conversational AI layer with face recognition, gesture detection, text-to-speech, and emotion-based behavior to create a more interactive assistant experience.

This repo is the fuller honors-project version of the app, with a GUI, camera-based interaction, and configurable emotional modes.

## Tech

- Python
- PyQt5
- Groq API
- OpenCV
- MediaPipe
- face-recognition
- gTTS

## Main Files

- `main.py` runs the desktop interface.
- `Grock_api.py` handles chatbot calls.
- `Grec.py` manages gesture and face-based interaction.
- `register_face.py` is used to add known users.
